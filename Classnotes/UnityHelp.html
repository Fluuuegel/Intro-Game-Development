Download and Installation

Unity download and install: work with the latest, all examples in class are based on Unity-2021.3.2f1 (LTS: Long Term Support releases):

  • https://unity3d.com/unity/qa/lts-releases. Under Long Term Support select: Unity 2021 LTS:
    • Download LTS Release 2021 LTS. Do not be overly concerned with the exact release, one of the LTS from 2019 and beyond will do.
    • Modules to install:
      • IDE: I will be using Visual Studio Code (VSC). With VSC, here is the debugger extension for Unity. You are free to use anything IDE you like, the default IDE for Unity is Microsoft Visual Studio Community, which works very well also.
      • WebGL Build Support: Please make sure to install this module for our playtesting (for part-2 of this class will be based on WebGL builds. Additionally, if you are running on a Mac, please also install Windows Build Support such that you can build for PC EXE and the TA can run your assignment submissions.
      • Documentation: I would advice installing this as well.
